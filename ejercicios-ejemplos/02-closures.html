## Closure.- Permite restringir el acceso a a un determinado valor, no podiendose leer y/o modificar directamente , sirve como uso de/o para variables privadas.Vuelve privadas a las variables con el uso de funciones. Permite encapsular valores.

Son funciones que regresan una función o un objeto con funciones que mantienen las variables que fueron declaradas fuera de su scope.

Los closures nos sirven para tener algo parecido a variables privadas, característica que no tiene JavaScript por default. Es decir encapsulan variables que no pueden ser modificadas directamente por otros objetos, sólo por funciones pertenecientes al mismo
